## Project setup

```bash
$ npm install
```

## Env variables set up

```bash
create .env file with all the vars displayed in AppModule

create docker.env with configuration of db
```

## Create container

```bash
$ docker-compose up
```




## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Run manual tests


Open localhost:<YOUR-HOST> in browser and see the playground with all the types and queries/mutations examples
